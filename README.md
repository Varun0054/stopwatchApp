# Stopwatch App

A sleek and modern stopwatch application for Android, designed with a beautiful, dark-themed user interface. This app provides precise time measurement up to the millisecond, lap recording functionality, and an elegant analog-style display.

## Features

- **Accurate Timing:** Measures elapsed time in `HH:MM:SS:ms` format with high precision.
- **Analog & Digital Display:** Features both a classic digital time readout and a custom-drawn analog stopwatch face.
- **Start, Stop, and Resume:** Control the timer with intuitive start and stop buttons. The timer can be resumed from its paused state.
- **Lap Functionality:** Record and display multiple lap times in a clean, scrollable list. The list shows the lap number, split time, and total time for each lap.
- **Reset:** A dedicated button to clear the current timer and all recorded laps.
- **Modern UI:** A beautiful, dark-themed design inspired by modern aesthetics, providing a great user experience.
- **Custom Views:** Includes a custom-drawn `AnalogStopwatch` view to visualize the elapsed time.

## Tech Stack & Dependencies

- **UI:** Android XML Layouts, Custom Views
- **Architecture:** Standard Android Activity Lifecycle

## How to Set Up

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Varun0054/stopwatchApp.git
    ```
2.  **Open in Android Studio:**
    - Open Android Studio.
    - Click on `File` > `Open`.
    - Navigate to the cloned repository folder and select it.
3.  **Sync Gradle:**
    - Let Android Studio sync the project with the Gradle files.
4.  **Run the App:**
    - Select an emulator or connect a physical device.
    - Click the 'Run' button (▶️) in Android Studio.
